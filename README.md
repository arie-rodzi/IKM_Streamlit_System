# Sistem Pintar IKM Nasional

## Cara guna
1. Extract ZIP.
2. Install: `pip install -r requirements.txt`
3. Run: `streamlit run app.py`

## Fungsi utama
- Dashboard nasional/negeri/daerah
- Pengiraan IKM dan IKM komposit
- Analisis indikator dan graf
- Analisis media sosial / big data
- Cadangan intervensi ikut lokasi
- Modul amaran awal
- Timeline 18 bulan

Data Excel ialah simulasi sintetik.
